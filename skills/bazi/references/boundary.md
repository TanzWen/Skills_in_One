# 边界情况与定盘校核

子平命理对**时辰精度**极敏感。以下边界处理不当会导致整篇分析全错。

## 一、立春边界（年柱划分）

**核心规则：年柱以「立春」交节为分界，不是正月初一。**

- 香港天文台与台湾农民历的官方干支纪年均采用"立春换年"
- 立春日时刻可精确到分秒，须查权威历表（如香港天文台、紫金山天文台）
- 立春**之前**出生 → 仍属上一年干支
- 立春**之后**出生 → 已属新年干支

**示例**：2018 年立春为 02 月 04 日 **05:28**。
- 2018-02-04 04:50 出生 → 仍属丁酉年（甲寅月）
- 2018-02-04 06:10 出生 → 已入戊戌年（甲寅月）

> 农历"正月初一"换年的口诀**只用于民俗**，不用于命理排盘。

## 二、月柱边界（节气划分）

月柱以**十二节**（非中气）为换月点：

| 月份 | 节气 |
|------|------|
| 寅月 | 立春 |
| 卯月 | 惊蛰 |
| 辰月 | 清明 |
| 巳月 | 立夏 |
| 午月 | 芒种 |
| 未月 | 小暑 |
| 申月 | 立秋 |
| 酉月 | 白露 |
| 戌月 | 寒露 |
| 亥月 | 立冬 |
| 子月 | 大雪 |
| 丑月 | 小寒 |

节气前后 24 小时内出生者，必须查交节精确时刻。

## 三、时柱边界（子时争议）

**核心争议**：子时（23:00–01:00）跨日，到底属当日还是次日？

| 流派 | 做法 |
|------|------|
| **早子时 / 夜子时分法**（主流） | 23:00–24:00 用次日日干配子时（夜子时）；00:00–01:00 用当日日干配子时（早子时） |
| **整时换日法**（少数） | 23:00 即换日，整个子时都用次日日干 |
| **传统派** | 子时不分早晚，统一用次日日干 |

**本 skill 默认采用"早夜子时分法"**，与香港天文台、台湾命理界多数主张一致。遇到子时出生，必须向命主说明所用流派。

## 四、真太阳时校核

公历时间是**钟表时间**（统一时区），命理用的是**真太阳时**（太阳实际位置）。两者差异来自：

1. **经度时差**：每偏离时区中央经线 1° = 4 分钟
   - 例：北京（116°E）相对东经 120°（北京时间标准）偏西 4° = 慢 16 分钟
2. **均时差**（Equation of Time）：地球公转椭圆轨道造成，全年波动 ±16 分钟

**校核公式**：
```
真太阳时 = 钟表时间 - 经度时差 ± 均时差
```

**何时必须校核**：
- 出生时间靠近时辰交界（前后 30 分钟内）
- 出生地远离时区中央经线（如新疆使用北京时间）
- 历史时期采用过夏令时（如中国 1986–1991 年实行过夏令时，须扣回 1 小时）

## 五、夏令时（DST）

中国大陆 1986–1991 年曾实行夏令时（4 月中旬至 9 月中旬钟拨快 1 小时）。该时段出生者需将钟表时间**减 1 小时**才是标准时间。

## 六、跨日 / 跨月 / 跨年的处理流程

遇到边界八字，必须按下列顺序询问 / 校核：

1. **公历日期与精确出生时间**（含分钟）
2. **出生地点**（用于真太阳时校核）
3. **是否处于夏令时期**（1986–1991 年中国大陆出生者）
4. **该时刻是否处于立春、节气、子时三大边界附近**
5. 如有任一边界 → 查权威历表得到精确交节时刻，重新排盘

**口诀**：
> **「年看立春，月看节气，时看子时，远地看真太阳时。」**

边界情况无法确认时，应明确告知命主"此盘存在边界争议"，并给出两种可能的排盘结果，请命主以人生经历反推哪一组更符合实际。
