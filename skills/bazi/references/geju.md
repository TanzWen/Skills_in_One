# 格局判断详解

格局是命局的「人生主题主线」。子平正法：**先看是否成立特殊格局，再以月令取普通格局**。

## 一、特殊格局（优先级最高，宁可不取，不可乱取）

满足以下任一条件且**全无破坏**方可取特殊格局。任何一点克制即不成格。

### 1. 从旺格
- 日主极旺，八字中无官杀
- 或官杀极微且被合化
- 比劫印星满盘

### 2. 从弱格（从财 / 从杀 / 从儿格）
- 日主极弱，无印无比劫
- 全盘均为财星 / 官杀 / 食伤之一
- 从财：满盘财星；从杀：满盘官杀；从儿：满盘食伤

### 3. 化气格
- 日主与月干（或时干）相合：甲己化土、乙庚化金、丙辛化水、丁壬化木、戊癸化火
- **且月令为所化五行当令**
- 天干无克合之神（如丁壬化木，需无庚辛克木；甲己化土，需无甲乙克土）

### 4. 专旺格（一行得气）
- **曲直格**（木一气）：日主木，地支寅卯辰或亥卯未
- **炎上格**（火一气）：日主火，地支巳午未或寅午戌
- **稼穑格**（土一气）：日主土，地支辰戌丑未全
- **从革格**（金一气）：日主金，地支申酉戌或巳酉丑
- **润下格**（水一气）：日主水，地支亥子丑或申子辰

> ⚠️ 专旺格条件极严苛，须四柱地支构成方局或三合局，且无冲克。

## 二、普通格局（月令取格）

### 取格五步法

1. 查月支【地支藏干表】，得**主气、中气、余气**
2. 检查**主气**对应的十神是否透出天干 → 透出则取此为格
3. 若主气不透，看**中气**是否透出 → 透出则取中气为格
4. 若均不透，**以月令主气定格**（气藏在支中）
5. **四墓月（辰戌丑未）**藏干较杂，以透干者优先取格

### 普通格局命名（依月令十神）

正官格、七杀格、正印格、偏印格、食神格、伤官格、正财格、偏财格

> **比劫透干不取格**：比劫当令时，视日主身旺，另取他神为格。

### 月令对照速查（日主×月支主气十神）

逻辑：先看月支主气，再以日主天干推十神，即得格。例：
- 甲日生于酉月 → 酉藏辛 → 辛克甲为正官 → **正官格**
- 庚日生于卯月 → 卯藏乙 → 庚克乙为正财 → **正财格**
- 壬日生于午月 → 午藏丁己 → 丁为壬之正财；若己土透干则取正官格
- 丙日生于寅月 → 寅藏甲丙戊 → 甲生丙为偏印 → **偏印格**

## 三、格局成败与高低

《子平真诠》核心论断：

- **顺用之神**（财、官、印、食）→ 喜生扶
- **逆用之神**（杀、伤、劫、刃）→ 喜克制

格局**「成」** 的条件：
- 用神（即格局星）有力，**透干通根**
- 有相神（帮助用神的神）护卫
- 忌神被制

格局**「破」** 的条件：
- 用神被冲克合化失力
- 喜神被忌神攻击且无救应
- 多个格局星混杂无主

> 「论命无以辰戌丑未即可格」——四墓月需更仔细辨析透藏。

## 四、常用复合格局名

- **食神制杀**：七杀格，食神透出制杀，富贵之格
- **伤官配印**：伤官格，印星透出制伤，主聪明显贵
- **伤官生财**：伤官格，财星透出，主富不主贵
- **财官印三全**：正官 + 正财 + 正印俱透有根，主大富大贵
- **杀印相生**：七杀化为印生身，主权威
- **官印相生**：正官生印、印生身，主名声地位

## 五、判格常见歧义与对策

| 歧义 | 处理 |
|------|------|
| 月令主气、中气均透出 | 以**主气**为本格，中气作辅；若中气得局更旺，可论复合格局 |
| 月令藏干均不透 | 以月令主气定格，但格局力量较弱，需配合调候、用神补强 |
| 比劫透干当令 | **不取比劫格**，转看月令其他藏干透出情况 |
| 化神不真（合而不化） | 不取化气格，仍按普通日主论 |
| 似从非从 | 严格审视有无微根、印星余气；有一线生机则不能从，转扶抑取用 |

格局确定后，所有后续分析（用神、喜忌、宫位、大运）都围绕这条主线展开。
