## 三、十神计算规则（以日主天干为基准）

- 同我者：同五行同阴阳 → **比肩**；同五行异阴阳 → **劫财**
- 我生者：我所生，同阴阳 → **食神**；异阴阳 → **伤官**
- 我克者：我所克，异阴阳 → **正财**；同阴阳 → **偏财**
- 克我者：克我者，异阴阳 → **正官**；同阴阳 → **七杀**
- 生我者：生我者，异阴阳 → **正印**；同阴阳 → **偏印**

> 阴阳：甲丙戊庚壬为阳，乙丁己辛癸为阴
