# 用神取法详解

用神是命局的「主治方向」，子平正法用神有严格优先级。**用神须透干通根方有力**，无根之用神为虚浮。

## 一、用神三大类与优先级

```
调候用神（寒暖燥湿） → 扶抑用神（强弱平衡） → 通关用神（两神对峙）
```

实战时三者并非互斥，常并行综合。原则：

- **调候优先**：极寒极热之月，先解决调候问题，否则其他用神也难施展
- **调候与扶抑一致** → 用神力量倍增，命局清纯
- **调候与扶抑冲突** → 需权衡轻重，明确说明取舍依据

## 二、调候用神

按 [tables.md](tables.md#四调候用神速查表按日主月令) 的【调候用神速查表】，按"日主×月令"查得。

**调候大原则：**
- 生于极寒之月（亥子丑） → 优先取**暖**（火）
- 生于极热之月（巳午未） → 优先取**润**（水）
- 燥土之月（戌未）忌过燥，喜湿润之神
- 湿土之月（辰丑）忌过湿，喜温暖之神

## 三、扶抑用神

| 日主状态 | 用神方向 | 具体十神 |
|----------|----------|----------|
| **身强** | 克泄耗 | 官杀（克）、食伤（泄）、财（耗） |
| **身弱** | 生扶 | 印星（生）、比劫（帮） |

**取舍次序：**
- 身强：以**克**为先，克力不足再取**泄**，最后取**耗**
- 身弱：以**印**为先，印弱再取**比劫**
- **印重身旺**：忌再生印，取财损印
- **比劫太重**：取官杀制比劫，无官杀则取食伤泄秀

## 四、通关用神

两神对峙、强度相当时，取中间通关之神调和。

| 对峙双方 | 通关用神 | 原理 |
|----------|----------|------|
| 财 vs 比劫 | **食伤** | 比劫生食伤，食伤生财 |
| 官杀 vs 食伤 | **印星** | 食伤生财克印，印生身制食伤 |
| 印 vs 财 | **官杀** | 财生官，官生印 |
| 杀 vs 印 | （已通关） | 杀印相生，本身即通关结构 |

## 五、用神「有力 / 无力」判定

| 条件 | 力量 |
|------|------|
| 透干且地支通根 | **最有力** |
| 透干无根 | 虚浮，吉处藏凶 |
| 藏支不透 | 力量在内，外不显达 |
| 被合化 | 用神被合住，须看合化后归属 |
| 被冲克 | 用神受损，需相神救应 |

**用神受损时的救应：**
- 用神被冲 → 看有无合住冲神
- 用神被克 → 看有无相神制克神
- 用神被合 → 看合化是否仍归用神方

## 六、喜忌神判定逻辑

确定用神后推导：

| 类别 | 定义 | 判定方法 |
|------|------|----------|
| **用神** | 命局最需要的五行/十神 | 按上述三法确定 |
| **喜神** | 帮助用神、引出用神的五行 | 生用神者，或与用神同类 |
| **忌神** | 克害用神的五行 | 克用神者，或助旺已旺之日主 |
| **仇神** | 帮助忌神的五行 | 生忌神者，使忌神更强 |
| **闲神** | 对命局影响不大 | 不生不克用神忌神者 |

> ⚠️ **喜忌必须结合具体格局推导**，不可套用"身强忌印""身弱喜印"等简化口诀而忽略格局本身的用神需求。
>
> 例：七杀格身强，官杀本为忌；但若以官杀为格局用神（食神制杀格、杀印相生格），则官杀反为喜。

## 七、用神取法常见陷阱

| 陷阱 | 正解 |
|------|------|
| 见身弱就用印 | 须看月令、有无调候冲突；冬日水旺身弱者，反需火调候 |
| 见身强就用财 | 须看比劫力量；比劫过重夺财，需先以食伤化比劫 |
| 见伤官就用印 | "伤官配印"成立条件：身弱、伤官旺；身强伤官旺则取财，不取印 |
| 调候忽略扶抑 | 调候用神若日主不堪承受，需先扶日主再用调候 |
| 用神被合 | 必须追查合化后的归属，未必失效 |

用神确定后，整张命盘的吉凶判读、大运流年好坏、人生主线走向都以此为准绳。
