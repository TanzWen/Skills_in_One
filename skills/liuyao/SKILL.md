## 工作流（分层灵活，非线性）

不要机械逐项过流程。正确节奏是：**（必要时先排盘）→ 识盘 → 问情厘清占问 → 定用神 → 看旺衰 → 察动变 → 综合断 → 印证校准 → 定应期**。哪一层信息足够下结论就下结论，不必所有层都写满；哪一层有疑点就深入该层。问情与印证不是独立一环，而是贯穿全程的两个动作——**问情在前（厘清占问、辅助锁定用神），印证在后（断后请命主确认可验证细节，对则增信、错则复盘）**。

### 阶段一·半：问情与印证（结合命主实情）

六爻定吉凶靠卦理，但**应事（落到具体人事）须有现实坐标**。同一卦象，命主背景不同，断语轻重、应期长短、趋避建议都不同。所以解卦要适度向命主问情，并在断后请其印证——但**必须守住「卦象为体、实情为参」的底线，绝不靠套话钓答案再装作卦说的**。

#### 何时问、问什么

| 时机 | 目的 | 该问什么 |
|-----|------|---------|
| **定用神前（问情）** | 厘清占问、锁定用神 | 占的是谁（自己/某关系人）、问哪一层（如问工作是入职/在职/升迁/待遇）、关心的核心是什么 |
| **卦象有歧义时（问情）** | 现实细节辅助定盘 | 用神多现择主、用神不上卦、多爻动难定主次时，问相关客观背景（事态进展到哪一步、涉及几人、身份阶段）帮助锁定 |
| **落应期时（问情）** | 把卦上应期对到真实时间线 | 已发生到哪一步、有无 deadline 或关键节点、起算点是何时 |
| **断卦后（印证）** | 校准判断、增信或复盘 | 主动说出几个**可验证的卦象细节**（如方位、人物特征、事态起伏、近期某事），请命主确认 |

**只取客观事实**：命主的身份、年龄、关系、事态阶段、时间节点等**客观背景**可纳入判断；命主的**情绪、期望、主观猜测、希望听到的结果**只作了解，**不得反过来左右用神旺衰的客观判读**。

#### 怎么问（防污染四铁律）

1. **先卦后问**：能从卦上读出的，先读出来再以提问确认；提问用于**确认**而非**套取**。避免「先问出答案，再倒装成卦象」的冷读套路。
2. **不诱导**：不问带吉凶倾向、会暗示答案的引导性问题（如「最近是不是破财了？」应改为先据兄弟爻发动断出破财之象，再问「卦上见近期有耗财，可有此事？」）。
3. **印证落空就复盘**：说出的细节命主反馈为「不对」，不可硬圆——回头查用神是否取错、旺衰是否评错、动爻主次是否定反，修正后再断。这正是好卦师与庸师之分。
4. **一次问到位**：把要问的客观信息**集中、精炼地一并问**，不要反复盘问像在套情报，更不要因命主一句反馈就反复改断。

#### 印证与结论的关系

- 印证**相符** → 增强该断的置信度，可在此基础上深入应期与细节。
- 印证**部分相符** → 保留相符部分，对不符处回到对应卦理层级复查。
- 印证**完全不符** → 多半是用神取错或装盘有误，回阶段一、二重来；必要时说明可能需另占（见「多占与使用边界」）。

> 注意：印证是为了**校准卦理判读**，不是用命主的反馈替卦下结论。卦象与命主实情冲突且无法调和时，如实指出张力、存疑，不强行统一。

**取用原则（优先序）**
1. **唯一性优先**：某六亲只现一爻，该爻即用神；多现则取发动者、临世应者、临日月者
2. **最亲者优先**：问具体之人（如"我老婆"），对应六亲优于通用六亲
3. **切题优先**：问何事取何神（问工作机会取官鬼，问待遇多少取妻财），不可混淆
4. **一事一用神**：问"这工作能做多久且待遇如何"拆两问——前看官鬼，后看妻财

**用神多现的过滤**：卦中同一六亲出现两爻以上时，依"**舍其闲废，用其用事**"——优先取**临世应、发动、或带特殊时空标记（逢冲破/空亡/合绊/临日月）**的那一爻为主用神，余者作背景参考。择主用是健康、家宅等多重六亲卦的常见关键（参 `references/cases.md` 案例三）。

**四神网络**：用神锁定后自动生成四节点动力网络——原神（生用神，助力资源）、忌神（克用神，阻力病灶）、仇神（克原神且生忌神，间接破局之帮凶）。断卦实质就是量化这四方在日月建下的博弈。

**伏神**：用神不上卦时从本宫首卦同爻位取伏神（详见阶段五与 `references/paipan.md`）。

### 阶段三：四柱权衡（旺衰评估）

这是**全盘最要命的一步**。旺衰搞错，后面全盘皆错。评估顺序：**月建 → 日辰 → 旬空 → 动爻生克 → 综合定级**。

#### 1. 月建（旺相休囚死）

月建定一月之纲领，按爻支五行与月建五行的关系赋予能量级别：

| 关系 | 状态 | 力度 | 含义 |
|-----|------|-----|------|
| 与月建比和（同五行） | **旺** | +++ | 当令峰值 |
| 受月建所生 | **相** | ++ | 得环境强助、后劲足 |
| 生月建（泄气输出） | **休** | − | 退居二线、消耗 |
| 克月建（逆抗） | **囚** | −− | 被环境围困 |
| 受月建所克 | **死** | −−− | 遭致命打击、生机断 |

**月破**：爻支被月建相冲（如寅月卦中申爻）。月破之爻当月如朽木死灰，丧失生克力。判断要小心：
- 用神休囚又月破 → 真破，永不能用，主凶
- 用神旺相且发动又月破 → 暂破，出月或逢合日可用
- 出月之后月破自解

#### 2. 日辰（六爻之主宰）

日辰无视季节、为最敏锐的触发器：

| 关系 | 作用 |
|-----|------|
| 日辰生合用神 | 暗旺——纵月建休囚，亦得突如其来的微观提振 |
| 日辰同五行 | 比和助力 |
| 日辰克用神 | −1~−2 |
| 日辰冲用神（用神静而旺/有生扶）| **暗动**：静爻被冲而起，力同明动 |
| 日辰冲用神（用神静而休囚死绝无扶）| **日破**：枯木遭风、瞬间摧折，比旬空更无力 |
| 日辰冲用神（用神动）| 旺者冲实不散，衰者冲散 |
| 日辰合用神 | 合住暂停 |

**暗动是六爻特有之妙**：旺相静爻被日冲即变暗动，力如明动，但方向略异（明动主主动作为，暗动主被动遭遇、隐情显露）。是否暗动取决于该静爻的底蕴——旺则暗动，衰则日破。

#### 3. 旬空（真假空亡之辨）

以占日天干起旬，每旬空二支（查 `references/lookup.md`）。空有真假，不可见空即凶断：

| 空亡类型 | 判定 | 含义 |
|---------|------|-----|
| 旺空（假空） | 用神旺相而空 | 时机未到，出空之日/逢冲之日即应，吉事延后而成 |
| 衰空（真空） | 用神休囚死绝而安静而空 | 事物真实缺失，不能用，时移亦难复苏（凶事可解） |
| 动空 | 用神发动而空 | **必不空**：动则实，冲空填实之日应 |
| 冲空 | 日辰冲空爻 | 填实，空爻被冲反生发 |
| 合空 | 日月合空爻 | 暂合住，合破之日出空 |

#### 4. 动爻对用神的生克（含变爻）

动爻生用神（助）、克用神（伤，忌神发动最忌）、冲用神（静用神被冲反动、动用神被冲为散）、合用神（合住暂停）、与用神相化（看变爻：化进/退、回头生/克、化空墓绝——见阶段四）。

#### 5. 综合旺衰定级

给用神一个明确等级，并对忌神、原神同样评级：

| 级别 | 特征 | 结论倾向 |
|-----|------|---------|
| **极旺** | 月生日扶、动爻来生、不空不破 | 吉事必成，凶事必现 |
| **旺相** | 月日其一得令、无大克伤 | 事可成 |
| **平和** | 月日不生不克、无明显助泄 | 成败五五，看动爻 |
| **休囚** | 月日不扶、轻克轻泄 | 事难成但未绝 |
| **衰绝** | 月破日冲、真空入墓、忌神旺动来克 | 不成，或凶 |

忌神旺动→灾至；忌神休囚或逢克→灾减；原神旺动→贵人助；原神空破入墓→用神无根。

### 阶段四：动变演算

动爻决定事情发生，变爻决定结果走向。每个动爻都要单独推敲。

**动爻本位单向性（关键规则）**：变爻是动爻自身阴阳突变所生，能量传递严格**本位单向**——**变爻只回头作用于产生它的那个本位动爻，绝不跨爻位去生克卦中其他动爻或静爻**。判"回头生/克"只看动爻与其自身变爻这一对，不可让甲爻的变爻去生克乙爻。

#### 动爻数目之象

| 动爻数 | 常见意象 | 爻辞权重 |
|-------|---------|---------|
| 六爻安静 | 看日辰暗动与世应生克；静卦不必硬找变化 | 先看世用与时令 |
| 独发一爻 | **独发有力**，断卦最关键线索 | 重看本爻爻辞 |
| 两爻动 | 看是否同五行（助）、相克、相冲 | 看主次 |
| 三爻动 | 看是否构成三合局 | 以生克为主 |
| 四五爻动 | 主多头、纷乱、反复 | 爻辞作校验 |
| 六爻尽发 | 极动主大变、大事或失控 | 以生克为主 |

口诀：**一动重爻意，二动看主次，三动以上重生克，静卦先看世用与时令。** 独静（五动一静）则该静爻亦为主事关键。

#### 化神关系

| 化法 | 判定 | 含义 |
|-----|------|-----|
| **化进神** | 同五行按长生顺序前进（寅化卯、申化酉……查 lookup.md） | 力增、事进 |
| **化退神** | 同五行后退（卯化寅、酉化申……） | 力减、事退 |
| **化回头生** | 变爻生本爻（如木动化水） | 绝处逢生，休囚者亦可逆转 |
| **化回头克** | 变爻克本爻（如金动化火） | 本爻反被所变伤，自取灭亡，大凶 |
| **化空** | 动爻化出空亡爻 | 事中途停顿，待出空 |
| **化破** | 动爻化出月破爻 | 变化失效，事成空 |
| **化墓** | 动爻化出自身之墓 | 事被埋藏、人事陷滞 |
| **化绝** | 动爻化出绝地 | 事无生机 |
| **化长生** | 动爻化出长生地 | 事得延续 |

**伏吟**：动爻变出之爻与本爻地支相同（甲子化甲子），主呻吟叹息、重复、原地打转。
**反吟**：动爻变出之爻与本爻相冲（子化午、寅化申），主反复变化、事与愿违、去而复来。内卦反吟多主自家，外卦多主外事。

#### 合冲局与高级组合

**三合局**（任二爻动一爻得日月拱合即成局，亦可全动成局）：亥卯未木、寅午戌火、申子辰水、巳酉丑金。用神入吉局大吉，入忌神之局反凶。

**六合**：子丑、寅亥、卯戌、辰酉、巳申、午未。爻动逢合主事合成或被牵制。

**六冲**：子午、丑未、寅申、卯酉、辰戌、巳亥。六合卦逢冲先成后散，六冲卦逢合先散后合，卦变六冲事中途而散。

**合处逢冲 / 冲处逢合**：卦中已合之事，逢冲支则被冲散（应期应之）；卦中已冲之事，逢合支则复合。其中"**贪合忘冲**"：两爻正相冲时，其一被第三方强力（如日建）合住，则冲力瞬间瓦解、激烈冲突得调停——绝处逢生。

**贪生忘克（救应的高级机制）**：忌神发动本欲克用神，但若原神同时发动且在忌神与用神之间，则忌神之力被原神"截胡"——忌神生原神、原神再生用神，形成"忌神→原神→用神"的能量闭环。破坏力反被转化为推动力，此时不凶反吉。判忌神动时，**务必先查原神是否在场承接**，否则会把"有救之卦"误断为凶。

#### 动爻优先级

多动爻时优先看与用神直接相关者：①用神自身动（看化神走向）②忌神动（定灾，并查贪生忘克）③原神动（定助力）④仇神动（间接影响）⑤其他爻（看象意与六神）。

### 阶段五：伏神飞神（用神不现时）

本卦缺某六亲时，用神伏于本宫首卦同爻位之下，伏神能否"得出"是关键：

| 伏神处境 | 结果 |
|---------|------|
| 飞神生伏神 | 提携，伏神可用 |
| 飞神克伏神 | 伏神受制，除非日月冲开飞神 |
| 伏神克飞神 | **伏神得出**，有力可用 |
| 伏神生飞神 | 泄气力弱 |
| 伏神与飞神比和 | 帮扶可用 |
| 飞神空亡 | 伏神得现 |
| 伏神旺相、日月生扶 | 可用 |
| 伏神休囚又被飞神克 | 不能用，其事无望 |

伏神应期：多应于冲开飞神之日、或伏神值日/值月。

### 阶段六：卦体与卦象

有些卦象本身带强烈倾向，不可忽视。

**六冲卦**（乾兑离震巽坎艮坤八纯卦，加无妄、大壮）：主**事散、变快、速战速决**。婚姻合作长久之事不利；疾病诉讼脱困求速反吉。
**六合卦**（否泰困节豫复贲旅）：主**事合、长久、和合**。婚姻合作有利；疾病脱困胶着难散。
**游魂卦**（八宫第七卦）：主游荡不定、神思恍惚、外出。占行人主在外、占病主精神恍惚。
**归魂卦**（八宫第八卦）：主归回、结束、内守。占行人主将归、占事主收尾。
**卦宫五行**：本卦所属宫（乾兑金、离火、震巽木、坎水、艮坤土）影响总气，用神与卦宫相生相助为佳。

**互卦、错卦、综卦**（辅参，不夺主）：互卦看中层结构与内情、错卦看反面、综卦看对方视角。只作侧面校验，绝不压过用神生克（取法见 `references/paipan.md`）。

### 阶段八：爻辞参证（辅校验，不夺生克）

本路线以纳甲生克为主，**周易卦辞爻辞为辅**，用来给生克结论做方向性校验与情境补充，而非另立一套判断：

- **动爻少（独发或两动）**：可较重发动爻的爻辞，与生克结论相互印证
- **动爻多（三动以上）**：以用神旺衰、世应、生克制化为主，爻辞退为校验
- **静卦**：先看世用与时令，卦辞作整体氛围参考

用法铁律：**象辞校断，不夺生克**。若爻辞吉而用神生克明显凶（或反之），以生克为准，并说明二者张力。切勿用一句经文压过满盘生克。

### 阶段九：综合断卦（决断树）

不要只给一句"吉"或"凶"。按权力层级推演，分四层下结论：

**决断树要点**：①先审用神在日月建的底层基础——若月建休囚、日建日破，纵多重原神发动亦"根基尽毁"，裁为凶；②再审动量传导——忌神旺动时查有无救应（贪生忘克、回头生）；③再审世应耦合——用神纵旺，若生应不生世，硕果归他人。

#### 第一层：成败吉凶定论
围绕用神给明确判断（成/不成/反复、吉/凶/平），**必须指明依据**（用神何等旺衰、谁生谁克、有无发动、化神如何）。示例："用神旺相发动化进神、日月来扶、无忌神动克，此事必成"；"用神休囚入墓、忌神旺动来克，原神虽有却空亡无力，此事不成且防损伤"。

#### 第二层：应期推算（病药原理 + 多路径交叉）
应期最考功力。底层是"**病药原理**"——找出当前阻碍（病），等能化解它的时点（药）即应期。按用神状态选路径：

| 用神状态（病） | 应期路径（药） |
|---------|---------|
| 旺相发动 | 合日应（动而逢合为止）或本支日 |
| 静而旺相 | 冲日应（静而逢冲为动） |
| 衰弱用神 | 逢生扶之日 |
| 旬空用神 | 出空之日（本支日）或冲空填实日 |
| 入墓用神 | 冲墓之日（如申金入丑墓，逢未日冲丑开墓） |
| 绝地用神 | 逢长生日 |
| 合住用神 | 冲开合之日 |
| 月破用神 | 出月或填实本支日 |
| 化回头克 | 被克之日或变爻当令之日 |
| 伏神受制 | 冲开飞神之日或伏神值日 |

时间尺度：近事以日支、中期以月建、远事以年支流年。**多路径验证**：一个应期最好两条以上路径支持；若路径相互矛盾，回头重审用神定位与旺衰。

#### 第三层：细节描摹
结合六神、卦宫、爻位给具体情境：爻位所主（初爻基层童幼、二爻家宅妻室、三爻兄弟门户、四爻外事职场、五爻君位上级、六爻宗庙远方）、方位（爻支所属，查 lookup.md）、人物特征（六神+六亲+旺衰）、事态起伏（合冲变化）。

#### 第四层：趋吉建议
给可执行方向：何方向有利（用神所临方位）、何时机更佳（用神旺相月日）、何事宜避（忌神方位与应期）、何行为可助（强原神、避忌神的实际动作）。

## 多占与使用边界

**多占（重复起卦）**：稳妥之道是**一时一事一卦为主**。若首卦关键用神不现、问题发生的时间已变、或问题本身已变，可另占新卦，但须重新说明缘由——不可抱着"摇到满意答案为止"的心态反复起卦，那会污染信息、自欺欺人。

**使用边界（务必向用户说明）**：六爻属中国传统术数，宜作**传统解释方法、思路参考与决策辅助**，而非现代实验科学意义上的可重复预测。**医疗、法律、重大投资、人身安全等高风险决策，不宜单独依赖占断**，应回到现实证据与专业意见。把六爻当"辅助思考"而非"最终裁决"，能显著降低决策偏差。

## 易错提醒

1. **用神取错**：问工作看官鬼还是妻财？——能否得到/在职/升迁看官鬼，待遇多少看妻财。男占婚财、女占婚官，切勿混用。
2. **月破当真破**：旺动之爻虽月破亦可用，待出月应。
3. **静爻当无用**：旺相静爻被日冲是暗动，力如动爻；休囚静爻被日冲才是日破。
4. **旬空一律凶断**：旺相发动之空是假空（待时即应），休囚死绝安静之空才是真空。
5. **动爻不看变爻**：动必有变，漏看变爻=漏掉走向，化回头克常主大凶。
6. **变爻跨位生克**：违反本位单向性——变爻只回头管本位动爻，不生克他爻。
7. **忌神动即断凶**：先查贪生忘克与回头生等救应，否则误断。
8. **只看本卦忽略变卦**：变卦是最终结果。本卦吉变卦凶先顺后逆，本卦凶变卦吉先艰后成。
9. **六神当主神**：六神是修饰性象意，旺衰生克才决定吉凶。
10. **忽略卦体**：六冲六合游魂归魂有强倾向，不可不顾。
11. **应期单一路径**：一个应期应由多条线索交叉验证。
12. **爻辞压生克**：辞象只作校验，不夺生克。
13. **不会排盘硬背断语**：盘装错则满盘皆错，不确定先核对、先排盘（读 paipan.md），勿"像懂实不懂"。
14. **不自检就下结论**：好卦师与庸师之分，在于肯不肯回头检查矛盾。
15. **套话钓答案**：靠提问套出实情再装作卦上读出——污染卦象。须先卦后问，提问只为确认（见阶段一·半）。
16. **被命主期望带偏**：命主的情绪、希望、主观猜测只作背景，不得左右用神旺衰的客观判读。
17. **印证落空硬圆**：说出的细节命主反馈不符，应回头复盘用神与旺衰，而非强行自圆其说。

## 输出风格

- **先亮结论再讲依据**，不要让用户在长篇推理末尾才看到判断
- **每条判断注明依据**（某爻某六亲某旺衰，受何生克）
- **数据化旺衰评估**（如"用神妻财巳火，月建寅木生之，日辰午火比和，又逢子孙辰土发动来生——极旺"）
- **应期给出具体月日或时段**，避免"不久"这类模糊说法
- **不能断的如实存疑**，不硬断
- **保留易学术语**，首次使用时稍加白话解释，让未入门用户也能跟上
- **语气要稳**：吉事不夸大，凶事不渲染，客观陈述并给可行建议
