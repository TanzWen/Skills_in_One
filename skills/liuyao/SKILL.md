## 分析步骤

### 一、卦象总述
- 本卦卦名及基本象意
- 变卦卦名及变化方向（如有动爻）
- 主卦与变卦的综合象意

### 二、世应分析
- 世爻：所在爻位、所持六亲、五行、旺衰状态
- 应爻：所在爻位、所持六亲、五行、旺衰状态
- 世应生克关系：世克应/应克世/世生应/应生世/世应比和——对应问事双方关系

### 三、用神确定与旺衰
根据占问事项确定用神：

| 占问事项 | 用神 |
|----------|------|
| 问自身健康 | 世爻 |
| 问官职仕途 | 官鬼 |
| 问财运求财 | 妻财 |
| 问婚姻感情 | 女问官鬼/男问妻财 |
| 问子女 | 子孙 |
| 问文书合同 | 父母 |
| 问官司诉讼 | 官鬼（自身）子孙（对方） |

- 用神所在爻位
- 用神旺衰：月建生克、日辰生克、动爻生克
- 用神是否入墓、空亡、冲克

### 四、忌神与仇神
- 克用神者为忌神——忌神状态
- 生忌神者为仇神——仇神状态
- 原神（生用神者）状态——是否有力

### 五、动爻变爻分析
逐一分析每个动爻：
- 动爻六亲及五行
- 变爻六亲及五行
- 动变关系（化进、化退、化绝、化墓、化回头生、化回头克）
- 对用神的影响

### 六、六神象意解读
- 青龙临何爻：主喜庆、文书、贵人
- 朱雀临何爻：主口舌、文字、争讼
- 勾陈临何爻：主迟滞、田土、纠纷
- 螣蛇临何爻：主虚惊、怪异、缠绕
- 白虎临何爻：主凶险、血光、官非
- 玄武临何爻：主暗昧、桃花、盗贼

重点关注六神与用神、世爻的组合象意。

### 七、伏神分析（如用神不现）
- 用神是否伏藏
- 飞神对伏神的生克关系
- 伏神能否出现

### 八、空亡与入墓
- 哪些爻空亡（日空、旬空）
- 空亡对用神/世爻/忌神的影响（旺空、衰空区别）
- 哪些爻入墓，影响如何

### 九、断卦结论

**针对占问事项给出明确判断：**

1. **事情成败/吉凶**：给出明确判断（成/不成，吉/凶/平）
2. **时间节点**：根据用神旺衰、动爻五行推断应期
   - 日化：当日应验
   - 月化：当月应验
   - 五行对应时间：木→春/寅卯，火→夏/巳午，金→秋/申酉，水→冬/亥子，土→四季末/辰戌丑未
3. **细节象意**：结合六神、卦象给出具体情境描述
4. **趋吉建议**：如何化解忌神、加强原神
