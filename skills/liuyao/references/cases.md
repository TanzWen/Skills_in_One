# 实战案例集（断卦推理示范）

以下四例为经典文献（《增删卜易》）案例复盘，重在展示**完整推理链**——取用神 → 判旺衰 → 看动变 → 定应期 → 下结论——而非记某句断语。读者应学其**推理骨架**：怎样从一盘信息推出"成/不成、何时应、为何"。

每例都遵循同一示范结构：**先亮结论 → 逐步依据 → 启示**。这正是输出时应有的样子。

---

## 案例一·婚姻：女占夫妻能否和好（需之大过）

| 项 | 内容 |
|---|---|
| 占问 | 已婚女性问：夫妻已不和，将来能否和好 |
| 本卦/变卦 | 需 → 大过 |
| 动爻 | 初爻、四爻动 |
| 取用神 | 女占夫，**官鬼为夫星（用神）**；世爻看自己；应位兼看对方 |

**结论：表面尚未立即决裂，实则外缘已成，终以生离收场——不是"会和好"，而是"分"。**

推理链：
1. **先定占类**——这是"女占夫"而非泛占婚，故以官鬼夫星为重，断不可错把妻财当主用神。一旦用神取错，满盘皆歪。
2. **看世爻**——世为子孙而发动。子孙乃克官（夫星）之神，世动伤夫，是婚姻关系中极强的不利信号：求测者自身之势正在动摇这段关系。
3. **看夫星**——官鬼寅木在卦中现形，说明关系并未即刻断绝，仍有夫星可寻。
4. **看应位**——应位兼作对方现势，又见合化生扶夫星之象，主**对方已另有情感牵连，外缘比原配关系更活跃**。
5. **看变卦**——由需（拖延、带伤的等待）变大过（结构失衡、难以久承），事态由"勉强维系"走向"承载崩塌"。
6. **定应期**——以夫星冲世与后续合化所触发的时点定期（原案断次年寅月生离）。

**启示**：婚姻卦最大的错误源头是**用神取错**——男占以妻财为用、女占以官鬼为用，必须先分名分占类。用神一错，后面全错。

## 案例二·事业：占会试能否及第（艮之益）

| 项 | 内容 |
|---|---|
| 占问 | 会试（考试）能否及第 |
| 本卦/变卦 | 艮 → 益 |
| 动爻 | 初爻、三爻、五爻动（三动） |
| 取用神 | 功名以**官鬼为用**；文章以父母为辅；世爻看本人 |

**结论：官星持世而得月令生扶，又得日辰触发暗动反生，文书不空不绝——高中有据。**

推理链：
1. **先取用神**——考试/功名先看"官、父、世"三者：官为功名主题、父为文章文凭、世为本人。
2. **看世与官**——官鬼寅木持世，本人之势与功名主题同频，是明显强信号。
3. **看月日**——卯月助寅木（月令生用），甲申日冲世。申冲寅本似克，但寅木旺相，冲而成**暗动**而非冲散。
4. **看合局**——原案断"申日会成财局，不惟不克世，反来生世"：外来触发不是破局，而是资源汇拢、反助功名。这是"**暗动反生**"的关键，若把申日之冲一概当凶就会断反。
5. **看多动结构**——三爻动之卦，**不宜死盯单条爻辞**，应以官父世的生克制化、日月触发为主（"三动以上重生克"）。
6. **看变卦**——艮（止、守）之益（利有攸往、增益推进），由"蓄势"转"升发"，节律相合。

**启示**：事业卦常见误差在于把"日辰冲用神"一律当凶。**旺相之爻被冲为暗动（甚至反生），衰弱之爻被冲才是日破**。冲的吉凶，先看用神底蕴。

## 案例三·健康：占父亲近病何时缓解（乾之小畜）

| 项 | 内容 |
|---|---|
| 占问 | 父亲近病，何时能缓解 |
| 本卦/变卦 | 乾 → 小畜 |
| 动爻 | 四爻动 |
| 取用神 | 问父病以**父母爻为用**；卦中父母多现，**择旺者**辰土为主用神 |

**结论：当前因暗动来克而加重；原神午火虽动却被合住、一时生扶不到位；待丑日冲开合局、午火得以生辰土，病势方退。**

推理链：
1. **取用神（用神多现的处理）**——卦中父母爻不止一处。不是见"父母"就随手拿一个，而要**舍闲取旺**：辰土父母临月建、最有根气，定为主病位。
2. **看病势**——申日冲起寅木为暗动，寅木克辰土（用神），是病情当前加重的直接原因。
3. **看救应（原神能否真到场）**——午火发动，本可生辰土（火生土，午火为原神）。但午化未、午未相合，午火被"**合住**"，名义上是援兵，实际到不了场。
4. **定应期（病药原理）**——病在"合住"。药在冲开：待丑日冲开未土，合解，午火得以畅生辰土，通关打开，病势退。原案果于丑日起床，应期与结构吻合。

**启示**：**有原神不等于有救。** 原神若被合住、空住、墓住，纵是援兵也到不了场。断"何时应"的底层逻辑是"病药原理"——找出当前阻碍（病），等能化解它的时点（药）即应期。

## 案例四·财运：占求财何时可得（火天大有·静卦）

| 项 | 内容 |
|---|---|
| 占问 | 求财，何时可得 |
| 本卦/变卦 | 大有（六爻安静，无变卦） |
| 动爻 | 0（静卦） |
| 取用神 | 求财取**妻财爻**为用 |

**结论：财来就我（财爻克世），此财必得；唯财爻当下旬空，非"永远没有"而是"目前未实"，待出空填实之日可得。**

推理链：
1. **取用神**——求财先看财爻，不先看外应断语。妻财寅木为用神。
2. **看世财关系**——财爻克世，乃"财来就我"之象，原案断"此财必得"。
3. **看空亡（真假空之辨）**——财爻当下旬空。但财爻寅月本旺，**旺空为假空**，是"目前未到位"而非"根本无财"。
4. **看月令**——寅月助寅木，财本旺。但旺不等于立刻兑现，还要等出空、填实。
5. **定应期（静卦的应期）**——静卦不必硬找变化，应期常由出空、值日、填实来定；断至甲寅日出空填实可得。

**启示**：**静卦不等于没信息，用神空亡不等于彻底无望。** 真正要分清的是"当下未实"（假空，待时即应）与"根本无望"（真空，休囚死绝之空）。静卦先看世用、空亡与时令，不必无中生有找变化。

## 案例共性总结（断卦四步骨架）

| 步骤 | 四例如何体现 |
|-----|------------|
| 取用神 | 女占夫取官、考试取官父、问父病取父、求财取财——**先定占类名分，再取六亲** |
| 判旺衰 | 月令生扶/休囚、暗动、真假空、合住——**旺衰定体，空破不可一律死断** |
| 看动变 | 世动伤夫、暗动反生、原神被合、静卦无变——**动为始变为终，原神能否真到场是关键** |
| 定应期 | 冲世之时、出空填实、冲开合局——**以病药原理找"何时能补上当前的漏洞"** |

四例反复印证一句：**好卦师与庸师之别，在于肯不肯辨清旺衰真假、肯不肯回头自检。**
