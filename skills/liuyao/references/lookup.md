## 旬空速查（按占日起旬）

每旬十日，必有两支无天干可配而落空。以占日干支定旬首，查该旬空二支：

| 旬首 | 该旬空支 |
|------|---------|
| 甲子旬（甲子～癸酉） | 戌、亥 |
| 甲戌旬（甲戌～癸未） | 申、酉 |
| 甲申旬（甲申～癸巳） | 午、未 |
| 甲午旬（甲午～癸卯） | 辰、巳 |
| 甲辰旬（甲辰～癸丑） | 寅、卯 |
| 甲寅旬（甲寅～癸亥） | 子、丑 |

## 刑冲合害

- **六冲**：子午、丑未、寅申、卯酉、辰戌、巳亥
- **六合**：子丑（合土）、寅亥（合木）、卯戌（合火）、辰酉（合金）、巳申（合水）、午未（日月之合）
- **三合局**：亥卯未木、寅午戌火、巳酉丑金、申子辰水
- **三刑**：寅巳申（恃势之刑）、丑戌未（无恩之刑）、子卯（无礼之刑）、辰午酉亥（自刑）
- **六害（穿）**：子未、丑午、寅巳、卯辰、申亥、酉戌

## 化进神 / 化退神速查

同五行地支按十二长生顺序前进为进神、后退为退神：

- **进神**：寅→卯、巳→午、申→酉、亥→子、丑→辰、辰→未、未→戌、戌→丑（力增、事进）
- **退神**：卯→寅、午→巳、酉→申、子→亥、辰→丑、丑→戌、戌→未、未→辰（力减、事退）
