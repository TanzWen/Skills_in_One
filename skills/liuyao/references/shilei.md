# 按事类断卦侧重

不同占问侧重完全不同。先按本表确认"用神是谁、忌神是谁、辅看什么"，再回 SKILL.md 主流程展开旺衰动变。

## 六亲的符号学外延

同一六亲在不同事类中指代灵活切换，这是六爻能断万事的核心机制。取用时务必落到**当前占问的具体语义**：

- **妻财**：钱财货物、利润；商业中为资金流/商品；婚姻（男占）为妻
- **官鬼**：官职功名、丈夫；疾病中为病邪病灶；诉讼中为案件/官府；商业中为约束/官方
- **父母**：长辈文书、房屋车船、学业；疾病中为医院病房；商业中为合同/牌照
- **子孙**：子女晚辈、医药、福神；商业中为产品研发/客户群/财源；诉讼中为和解消灾
- **兄弟**：兄弟同辈、同行竞争者、合伙人；财上为劫财耗损

## 问疾病

- **用神**：自己问己用世爻，问他人按关系取六亲用神
- **官鬼为病邪**：官鬼旺动克用神 → 病重；官鬼衰、空、被克 → 病轻或愈
- **子孙为医药**：子孙旺相发动 → 良医良药；子孙空亡休囚 → 求医无效
- **父母持世或父母动** → 主忧愁、服药不当
- **六冲卦利近病**（病去得快），**六合卦不利病**（缠绵难愈）；**久病逢冲反凶**（垮）
- **化绝、化墓、化回头克** → 大凶
- **应期**：用神逢冲日、忌神得令之日尤需警觉；缓解期看通关之日（参案例三）

## 问财

- **用神**：妻财；**兄弟为忌**（旺动主破财、竞争、合伙不利）；**子孙为财源**（旺动生财则源源而来）
- **世持财爻旺相** → 自有之财；**世克财爻** → 主动取财；**财爻持世而动** → 财动不稳、易得易失
- **财化退神** → 财渐消；**财化进神** → 财渐增
- **静卦求财**：先看世用、空亡、时令，不必硬找变化（参案例四）
- **应期**：财爻旺相者逢合日、动财逢生日、空财出空填实日应

## 问婚恋

- **男占妻财、女占官鬼**为用（取错则满盘皆歪，参案例一）
- **世应关系**：世应相生为和睦、相克为不合、比和为同心、世生应为迁就、应生世为得宠
- **世应空**：某一方空亡即该方无意
- **六合卦** → 可成且长久；**六冲卦** → 难成或成而速散
- **忌神发动**（男占见兄弟动、女占见子孙动） → 婚变之兆
- **应期**：六合卦以卦气合成之日应

## 问官职、求职、考试

- **用神**：官鬼（功名职务）；**父母为印绶文书**（旺相生官则有凭有据可得）
- **子孙为忌**（旺动克官 → 阻力大、被裁、不得升）
- **官鬼持世旺相** → 有官可得；**官鬼空破衰绝** → 不得
- **官鬼化进神** → 升迁有望；**化退神** → 降职失位
- 考试合同看父母为主、官鬼为辅（录取/批文）
- **多动之卦重生克**，不死盯单爻辞（参案例二）
- **应期**：官鬼得令之月、生扶之日

## 问行人

- **按关系定用神**（问夫看官、问友看兄弟）
- **游魂卦** → 在外游荡未归；**归魂卦** → 即将归回
- **用神动化回头生/化进神** → 归心切；**用神静且无动爻相扰** → 尚不动身
- **用神空亡** → 消息断；**用神入墓** → 被阻
- **应期**：用神出空日、冲日、长生日为归

## 问失物

- **用神**：妻财；**玄武所临爻**示盗贼方向（爻支方位）与特征（六神+六亲）
- **妻财旺相无动** → 财物尚在；**妻财空、动、化绝** → 物已远离
- **世爻克财** → 自己可寻回
- **应期**：财爻冲日、值日应

## 问诉讼

- **世为自己、应为对方**
- **官鬼为案件官府**（旺动克世 → 官非严重、对己不利）
- **子孙为和解消灾**（发动 → 有和解之望）；**父母为状纸文书**（动则文书频繁）
- **世应生克**：世克应我方占优、应克世对方占优
- **六冲卦利讼散**；**六合卦讼难了**
- **应期**：官鬼得令之日为审讯日，子孙出现之日为和解日

## 问天时晴雨

- **父母为雨雪**（旺动则雨多）；**妻财为晴**（旺动则晴朗）；**子孙为日月晴明**
- **兄弟为风**（动则风起）；**官鬼为雷电阴霾**（旺动则雷阵雨或阴霾）
- 看动爻与世应之生克判天气转变

## 问出行

- **世为自己、应为目的地**
- **父母为舟车工具**（动则舟车频繁或有事）；**兄弟为同伴**（旺动多同伴或破费）
- **官鬼为阻碍**（动克世 → 途中有阻）
- **世动** → 可行；**世静而应动** → 宜缓行
- **六冲卦** → 主变动宜变通；**六合卦** → 顺利
- **应期**：世动者其支之日、冲世者其支之日
