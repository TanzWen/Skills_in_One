# 三盘对应映射

分析前先建立三体系之间的领域 / 时间对应，这是综合分析的基础。

## 人生领域的三盘对应

| 人生领域 | 八字视角 | 紫微斗数视角 | 星盘视角 |
|---------|---------|------------|---------|
| 自我与性格 | 日主 + 十神格局 | 命宫主星 + 三方四正 | 上升星座 + 第一宫 + 日月 |
| 事业方向 | 官杀星 + 月令 | 事业宫（官禄宫） | 第十宫（MC） + 土星 |
| 财富模式 | 正偏财星 + 食伤生财 | 财帛宫 + 化禄落宫 | 第二宫 + 第八宫 + 金木 |
| 感情婚姻 | 男看财、女看官 + 日支 | 夫妻宫 + 红鸾天喜 | 第七宫 + 金星 + 月亮 |
| 学业考试 | 印星 + 食伤 | 父母宫 + 文昌文曲 | 第三宫 + 第九宫 + 水木 |
| 健康体质 | 日主强弱 + 忌神五行 | 疾厄宫 + 命宫 | 第六宫 + 第一宫 + 火星 |
| 人际贵人 | 印星 + 贵人神煞 | 交友宫 + 天魁天钺 | 第十一宫 + 木星 + 北交点 |
| 家庭根基 | 年柱月柱 + 印星 | 田宅宫 + 父母宫 | 第四宫（IC） + 月亮 |
| 精神内在 | 印星 + 食伤 + 日主阴阳 | 福德宫 | 第十二宫 + 海王 + 月亮 |

## 时间体系的对应

| 时间维度 | 八字 | 紫微斗数 | 星盘 |
|---------|------|---------|------|
| 十年运 | 大运（每步 10 年） | 大限（每步 10 年） | 外行星行运相位周期 |
| 年运 | 流年天干地支 | 流年（太岁入各宫） | 太阳弧 / 行运 / 太阳回归盘 |
| 月 / 日运 | 流月流日 | 流月流日 | 内行星行运相位 |
