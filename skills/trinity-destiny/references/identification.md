# 三张盘面识别清单

逐张仔细读取，信息量很大，不要遗漏。

**优先委派单体系 skill**：八字盘走 [[bazi]] skill、紫微盘走 [[ziwei]] skill，由它们完成识别与基础解读，本 skill 拿到结构化结论后专注做三盘交叉与综合。星盘无对应 skill，按下列清单自行识别。

## 八字盘面

- 出生年月日时（阳历 / 农历）、性别
- 四柱天干地支（年、月、日、时柱）
- 日主（日干）及其五行属性
- 十神配置：正官、偏官（七杀）、正印、偏印（枭神）、比肩、劫财、食神、伤官、正财、偏财
- 地支藏干
- 大运排列及起运年龄
- 当前所行大运
- 神煞标注（如有）
- 五行强弱 / 喜用神（如盘面已标注）

## 紫微斗数盘面

参考 [[ziwei]] 技能的识别标准：

- 命主星、身主星、五行局
- 十二宫位逐宫识别：宫名、地支、主星、辅星、煞星、杂曜、四化标记
- 命盘阴阳、大限顺逆行方向
- 大限信息（如可见）
- 生年四化落宫

## 西洋星盘

- 太阳星座、月亮星座、上升星座（ASC）
- 十大行星（日、月、水、金、火、木、土、天王、海王、冥王）的星座与宫位
- 重要相位：合相 0°、六分相 60°、四分相 90°、三分相 120°、对分相 180°
- 各宫宫头星座
- 北交点 / 南交点位置
- 凯龙星位置（如标注）
- 重要格局：大三角、T 三角、大十字等（如有）

## 识别完成后

用简表形式列出三盘核心信息，请命主确认。看不清的部分如实说明，不要猜。
