# 阶段一 —— 回溯过往（验证阶段）

目的：通过分析命主已发生的人生经历来验证三盘解读准确性，同时让命主建立信任。

根据命主当前年龄，推算已走过的大运 / 大限 / 行运周期，逐项回溯。

## 3.1 性格与自我认知

三盘交叉分析命主的核心性格特质：

- **八字**：日主五行属性、十神格局呈现的性格倾向
- **紫微**：命宫主星组合、三方四正星曜反映的性格面貌
- **星盘**：太阳 + 月亮 + 上升的三重性格结构

**汇总**：三体系对性格的描述是否一致？哪些特质被反复印证？有无矛盾？

## 3.2 学业阶段（约 6–22 岁）

定位这段时间对应的大运 / 大限 / 行运：

- **八字**：该步大运与印星、食伤的关系 → 学习能力与考试运
- **紫微**：大限走到何宫、大限四化影响 → 学业环境与成绩起伏
- **星盘**：土星、木星等慢行星在这一时期的关键相位 → 成长压力与机遇

给出具体时间点推测，例如："大约在 X 岁（对应 XXXX 年前后），三盘同时显示较大变动 / 转折 / 压力……"

## 3.3 情感经历

- **八字**：桃花运出现的大运 / 流年时段，正缘还是烂桃花
- **紫微**：夫妻宫与红鸾天喜的启动时间，大限命宫走入桃花宫位时期
- **星盘**：金星、火星行运触发第五宫 / 第七宫的时段

**重点推测**：初恋大约在什么年龄？有无明显感情挫折期？目前感情状态？

## 3.4 事业与财运轨迹

- **八字**：历步大运的财官运变化
- **紫微**：大限走入不同宫位时的事业财运特征
- **星盘**：木星、土星行运与第二 / 第六 / 第十宫的互动

**重点推测**：开始工作的大致时间、事业上升 / 低谷期、收入变化、是否有创业 / 跳槽等重大变动。

## 3.5 健康与重大事件

- **八字**：忌神五行对应身体部位，大运 / 流年冲克时的健康风险
- **紫微**：疾厄宫格局，大限经过煞星密集宫位的时期
- **星盘**：火星、土星、冥王星硬相位触发的时段

**重点推测**：是否有过伤病、手术或身体明显不适的时期？
