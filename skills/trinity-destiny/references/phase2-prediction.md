# 阶段二 —— 预测未来（预测阶段）

收到命主反馈后才能进入此阶段。先做三件事：

1. 简要回应反馈，说明哪些验证成功、哪些有偏差及可能原因
2. 根据反馈调整对三盘的信赖权重（例如：若紫微在过往验证中更准，未来预测中可更信赖紫微的指引）
3. 展开下列六方面的全面预测

## 4.1 当前运势总评（近 1–2 年）

三盘交叉定位当下：

- 八字当前大运 + 今年流年的天干地支组合
- 紫微当前大限宫位 + 流年宫位
- 星盘当前关键行运相位

综合判断：上升期 / 平稳期 / 调整期 / 低谷期？

## 4.2 事业前景（未来 3–10 年）

- **八字**：未来几步大运的官杀、印星、食伤变化 → 职业发展方向
- **紫微**：即将进入的大限宫位格局 → 事业机遇与挑战
- **星盘**：土星、木星、冥王星等慢行星未来的关键相位 → 转型节点

**具体到**：适合的行业方向、最佳发力时间窗口、需规避的风险期、是否适合创业 / 转行。

## 4.3 财运走势

- **八字**：未来大运与财星的关系
- **紫微**：财帛宫与化禄的流转
- **星盘**：木星行运周期与第二宫 / 第八宫

**具体到**：财运上升的年份、适合的求财方式（正财 / 偏财 / 投资）、破财风险期。

## 4.4 感情婚姻展望

- **八字**：正缘出现的大运 / 流年节点
- **紫微**：夫妻宫大限四化影响、红鸾天喜流年触发时段
- **星盘**：金星、木星行运第五宫 / 第七宫的时间

**具体到**：是否有明显婚姻节点？配偶大致特征（三盘综合画像）？婚后感情走势？

## 4.5 健康提醒

三盘交叉标注未来需注意的健康时段和对应身体部位，给出预防建议。

## 4.6 其他重要提示

根据命盘特征视情况覆盖（若命盘有明显指示）：

- 子女运
- 学业 / 进修时机（若命主年龄适用）
- 人际关系与贵人方位
- 居住 / 迁移方向
- 精神成长课题
