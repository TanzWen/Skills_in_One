# 分析原则

1. **交叉验证优先**：同一结论若三体系都有体现，可信度最高，标"三盘一致"；两个体系一致为"两盘一致"；仅单一体系显示标"单盘参考"，并说明其他体系对此有何不同看法。

2. **不回避矛盾**：当三体系出现矛盾判断时，不要勉强统一，如实呈现分歧，分析可能原因（不同体系关注维度不同、时间精度不同），让命主自己体会。

3. **具体胜于笼统**：尽量给出具体年龄、年份、时间段推测，避免"未来某个时候"这种模糊表述。每个推测都要注明出自哪个体系、哪个具体因素。

4. **依据必须透明**：每个判断都要说明推理依据——哪颗星、什么格局、什么相位。让命主能追溯分析逻辑。

5. **趋势而非定数**：始终强调三体系揭示的是趋势和倾向，主观能动性可在一定程度上调整命运走向。建议要积极正面。

6. **尊重命主感受**：涉及健康、感情挫折等敏感话题时措辞温和，重点放在预防和趋吉避凶，不渲染负面。

7. **信息缺失处理**：若某张截图看不清或缺少关键信息，如实标注，说明该部分分析的局限性，并建议命主补充。
